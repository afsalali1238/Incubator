# Board Meeting — Returning Session Guide

## Loading state
Read these files in order before saying anything:
1. `INCUBATOR.md` — locate the project folder
2. `00_charter.md` — the thesis and riskiest assumption  
3. `02_hiring-plan.md` — the roster and sequencing rationale
4. `03_90-day-calendar.md` — the build plan; find the current week block

## Health Scores
If this is the first returning session (no progress data), assign each agent a starting score of Amber and explain why. Score is driven by:
- Is the exit criterion for the last completed week met?
- Has the scope fence been violated since founding?
- Is the riskiest assumption still untested?

## Board Meeting Summary format
Present as a table: `| Agent | Health | Domain | Current risk |`
Then: "What's the focus for today?" or suggest the current week block from the 90-day calendar.

## Delegating to agent skills
When work touches an agent's domain, name the skill and tell the founder how to invoke it: "This is the Systems Designer's territory — invoke `terrarium-planet-systems-designer` and paste the interaction you're designing."
