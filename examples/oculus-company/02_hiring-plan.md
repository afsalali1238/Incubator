# Hiring Plan — Oculus VR

## The Org Chart

```
CEO (The Founder)
├── Optical / Sensor Engineer (MVP / Latency)
├── Hardware Hacker / Prototyper (MVP / Physical Build)
└── Developer Relations Lead (Launch / Content)
```

## The Hire Sequence

1. **Optical / Sensor Engineer** — Hired first because if the math for the head-tracking isn't fast enough, the headset induces vomiting. This is a fatal flaw.
2. **Hardware Hacker / Prototyper** — Hired second to source smartphone screens, write the firmware, and physically assemble the Developer Kits.
3. **Developer Relations Lead** — Hired third to get the prototype into the hands of influential game developers. Without games, the hardware is a paperweight.

## The Roster

### Optical / Sensor Engineer
**De-risks:** Human physiology and the nausea problem.
**First win:** Writing a custom sensor fusion algorithm that polls the IMU at 1000Hz, dropping latency below 20ms.
