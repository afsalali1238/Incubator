# 90-Day Build Calendar — Terrarium Planet

## Weeks 1–2 — Validate (Idea stage)
**Owner:** Systems Designer
- Paper prototype the 4 core elements (Water, Heat, Earth, Seed).
- Playtest the matrix with 3 casual strategy players using a spreadsheet or paper cards.
**Exit:** Players understand how elements combine without looking at a manual.

## Weeks 3–6 — Build the core loop (MVP stage)
**Owner:** Frontend/WebGL Engineer (+ Systems Designer reviewing)
- Build a 50x50 interactive grid in the browser.
- Implement the "tick" system (cellular automata rules).
- Add the 4 base elements with simple colors.
**Exit:** A human can click to drop elements and watch them interact in real-time at 60fps.

## Weeks 7–10 — Iterate toward evidence (MVP → Launch)
**Owner:** Systems Designer & Game Economist
- Implement the "Decay" system so the board clears over time.
- Add advanced emergent blocks (Moss, Steam, Glass).
- Build the "Unlock" progression (start with 2 elements, unlock the rest).
**Exit:** A playtester spends >15 minutes continuously messing with the grid.

## Weeks 11–12 — Launch readiness (Launch stage)
**Owner:** Game Economist & Frontend Engineer
- Final visual polish (shaders, particle effects).
- Add save/load states via local storage.
- Publish to itch.io or a web domain.
**Exit:** 100 anonymous web sessions with an average playtime over 5 minutes.
