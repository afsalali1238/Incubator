# Hiring Plan — Terrarium Planet

## The Org Chart

```
CEO (Afsal)
├── Systems Designer (MVP / Core Loop)
├── Frontend/WebGL Engineer (MVP / Client)
└── Game Economist / Balancer (Launch / Scale)
```

## The Hire Sequence

1. **Systems Designer** — Hired first because getting the elemental matrix right is the hardest and riskiest part. Lifecycle stage: Idea/MVP.
2. **Frontend/WebGL Engineer** — Hired second to make the cellular automata visually performant in the browser. Lifecycle stage: MVP.
3. **Game Economist / Balancer** — Hired third to pace the progression and unlock mechanics so players don't burn through content in an hour. Lifecycle stage: Launch.

## The Roster

### Systems Designer
**De-risks:** Core gameplay loop and interaction mechanics. 
**First win:** A playable text-based or grid-based prototype proving the basic 4 elements interact correctly without crashing.
