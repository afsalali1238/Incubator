# Hiring Plan — Gymshark

## The Org Chart

```
CEO (The Founder)
├── Community / Influencer Manager (MVP / Brand)
├── Apparel Designer / Screen Printer (MVP / Product)
└── E-commerce Ops Lead (Launch / Fulfillment)
```

## The Hire Sequence

1. **Community / Influencer Manager** — Hired first because without YouTube distribution, we just have a garage full of t-shirts. 
2. **Apparel Designer / Screen Printer** — Hired second to actually sew the taper and print the logos onto the blank garments.
3. **E-commerce Ops Lead** — Hired third to manage the Shopify store drop and print shipping labels when the traffic hits.

## The Roster

### Community / Influencer Manager
**De-risks:** Distribution and brand trust.
**First win:** Getting 3 fitness YouTubers with >50k subscribers to wear a Gymshark stringer in a workout video.
**Skill/Brief:** [team/community-manager.md](./team/community-manager.md)
